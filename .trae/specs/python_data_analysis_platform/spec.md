# Python数据分析AI训练平台 - 产品需求文档

## 概述
- **摘要**：基于Cloudflare免费资源，实现“3步认知+10个梯度项目+AI错题倒逼”的Python数据分析实操训练平台，零成本、零运维、无传统后端，打开浏览器即可使用。
- **目的**：为用户提供一个免费、便捷的Python数据分析学习平台，通过实际项目练习和AI指导，提升数据分析能力。
- **目标用户**：希望学习Python数据分析的开发者、学生和数据爱好者。

## 目标
- 基于Cloudflare免费套餐实现完整的Python数据分析训练平台
- 提供10个梯度项目，覆盖不同难度和应用场景
- 实现AI陪练功能，提供思路点拨和代码纠错
- 支持浏览器端Python代码运行，无需后端服务器
- 实现学习进度保存，确保用户体验连续性

## 非目标（范围外）
- 不使用任何付费服务或传统后端服务器
- 不支持大规模数据集处理（限制在浏览器能力范围内）
- 不提供多设备数据同步功能（MVP阶段）
- 不支持复杂的机器学习模型训练

## 背景与上下文
- 传统Python数据分析学习平台通常需要后端服务器支持，增加了部署和维护成本
- Cloudflare提供了丰富的免费资源，包括Pages、Workers、KV存储和AI Gateway
- Pyodide技术使得在浏览器中运行Python代码成为可能，为无后端架构提供了基础

## 功能需求
- **FR-1**：首页和学习引导模块，展示平台功能和学习路径
- **FR-2**：思维模型和行业争议模块，提供数据分析基础认知
- **FR-3**：10个梯度项目实现，包含数据集生成、代码编辑和运行
- **FR-4**：AI陪练功能，提供思路点拨和代码纠错
- **FR-5**：学习进度存储，保存用户代码和完成状态
- **FR-6**：在线代码编辑器，支持语法高亮和自动补全
- **FR-7**：图表渲染功能，支持Matplotlib生成的图表在前端显示

## 非功能需求
- **NFR-1**：性能要求：首屏加载时间<2秒，代码运行响应时间<3秒
- **NFR-2**：可靠性：系统稳定运行，无后端依赖
- **NFR-3**：安全性：AI API Key不暴露在前端，避免密钥泄露
- **NFR-4**：可用性：使用Cloudflare全球CDN，确保全球用户访问速度
- **NFR-5**：兼容性：支持主流浏览器（Chrome、Edge）

## 约束
- **技术**：基于Cloudflare免费套餐，使用React + TypeScript + Vite + Tailwind CSS + Pyodide
- **业务**：零成本部署和运维，不使用任何付费服务
- **依赖**：Cloudflare Pages、Workers、KV存储、AI Gateway、Pyodide

## 假设
- 用户拥有现代浏览器，支持WebAssembly（Pyodide运行需要）
- 用户网络连接稳定，能够加载Pyodide和相关资源
- Cloudflare免费套餐额度足够支撑MVP阶段的使用需求
- AI API（豆包/OpenAI）服务稳定可用

## 验收标准

### AC-1：前端部署成功
- **给定**：前端代码已提交到GitHub仓库
- **当**：配置Cloudflare Pages并部署
- **然后**：平台成功上线，可通过生成的域名访问
- **验证**：`programmatic`
- **备注**：部署过程应简单，无需复杂配置

### AC-2：Python代码运行
- **给定**：用户在代码编辑器中编写Python代码
- **当**：点击运行按钮
- **然后**：代码在浏览器中执行，显示运行结果和图表
- **验证**：`programmatic`
- **备注**：支持pandas、numpy、matplotlib等库

### AC-3：AI陪练功能
- **给定**：用户点击“思路点拨”或“代码纠错”按钮
- **当**：系统发送请求到Workers
- **然后**：AI返回相关指导，不直接提供完整代码
- **验证**：`human-judgment`
- **备注**：AI响应应符合提示词规范，语言简洁直白

### AC-4：学习进度保存
- **给定**：用户完成项目任务或编写代码
- **当**：系统自动保存进度
- **然后**：刷新页面后，代码和完成状态保持不变
- **验证**：`programmatic`
- **备注**：使用LocalStorage存储，无需后端数据库

### AC-5：性能指标
- **给定**：用户访问平台
- **当**：测量加载时间和代码运行时间
- **然后**：首屏加载时间<2秒，代码运行响应时间<3秒
- **验证**：`programmatic`
- **备注**：需要优化Pyodide加载和代码执行速度
